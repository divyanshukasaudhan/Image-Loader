🎨 Image Editor Application

A simple and user-friendly GUI-based Image Editor built using Python, Tkinter, OpenCV, and Pillow (PIL).
This application allows users to upload images and apply basic editing operations with real-time preview.

📌 Description

The Image Editor Application is designed to demonstrate the use of image processing and GUI development in Python.
It provides an interactive interface where users can easily modify image properties such as brightness, contrast, saturation, and blur.

🚀 Features
📂 Upload images from your system
🖼️ Display original and edited images side-by-side
🎚️ Adjust image properties:
Brightness
Contrast
Saturation
Blur
⚡ Real-time preview updates
🎨 Custom slider controls
🧑‍💻 Simple and clean user interface

🛠️ Technologies Used
Python 3
Tkinter – for GUI
OpenCV (cv2) – for image processing
Pillow (PIL) – for image handling

⚙️ Installation
Step 1: Install Python
Make sure Python 3.7 or higher is installed.

Step 2: Install Required Libraries
pip install opencv-python pillow

▶️ How to Run
python Main1.py


🎮 Usage
Click on Upload New Image
Select an image from your system
Adjust sliders:
Brightness
Contrast
Saturation
Blur
Click Apply Adjustments
View the edited image in the preview panel

🧠 How It Works
The image is loaded using OpenCV
Adjustments are applied using OpenCV functions
Saturation is modified using HSV color conversion
The final image is displayed using Pillow and Tkinter
